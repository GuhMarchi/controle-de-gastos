# 💰 Controle de Gastos Pessoais

Sistema de controle de gastos pessoais desenvolvido em Python com armazenamento em SQLite e JSON. Projeto desenvolvido durante meus estudos de Análise e Desenvolvimento de Sistemas.



👉 [github.com/GuhMarchi/controle-de-gastos](https://github.com/GuhMarchi/controle-de-gastos)

---

## 📌 Sobre o projeto

Sistema que roda no terminal permitindo ao usuário registrar, visualizar e analisar seus gastos do dia a dia, com agrupamento por categoria e persistência de dados em banco de dados SQLite e arquivo JSON.

---



- Adicionar gastos com descrição, valor e categoria
- Listar todos os gastos registrados
- Calcular o total de gastos
- Visualizar gastos agrupados por categoria
- Salvar gastos em arquivo JSON
- Persistência de dados com banco de dados SQLite

---

| Tecnologia | Uso |
|------------|-----|
| Python 3 | Linguagem principal |
| SQLite3 | Banco de dados local |
| JSON | Armazenamento secundário |

---



1. Clone o repositório:
```bash
git clone https://github.com/GuhMarchi/controle-de-gastos.git
```

2. Acesse a pasta:
```bash
cd controle-de-gastos
```

3. Execute o programa:
```bash
python3 Cpontroled_de_gastos.py
```

---



Ao executar o programa, o menu será exibido:

```
1 - Adicionar gastos
2 - Listar gastos
3 - Total Gastos
4 - Gastos por Categoria
5 - Salvar gastos
6 - Sair
```

Digite o número da opção desejada e pressione Enter.
